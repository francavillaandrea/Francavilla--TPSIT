## Come clonare la Repository

Per scaricare una copia locale del progetto e lavorarci, segui questi passaggi:

### 1. Installa Git
Se non hai ancora Git installato sul tuo PC, scaricalo dal sito ufficiale:
> [!IMPORTANT]
> **[Scarica Git per il tuo Sistema Operativo](https://git-scm.com/downloads)**

### 2. Clona via Terminale
Apri il tuo terminale (CMD, PowerShell o Git Bash) e digita i seguenti comandi:

```bash
# Spostati nella cartella dove vuoi salvare il progetto
cd Desktop
```
```bash
# Clona la repository
git clone https://github.com/francavillaandrea/Francavilla--TPSIT.git
```
```bash
# Entra nella cartella del progetto
cd Francavilla--TPSIT
```
